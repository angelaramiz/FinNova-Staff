Referencia de ecuaciones — GJR-GARCH + MLE + Tests
GJR-GARCH(1,1) — Glosten, Jagannathan, Runkle (1993)
σ²ₜ = ω + α·ε²ₜ₋₁ + γ·ε²ₜ₋₁·𝟙[εₜ₋₁<0] + β·σ²ₜ₋₁
γ > 0 → leverage effect (caídas generan más vol)
γ captura la asimetría: las malas noticias (εₜ₋₁ < 0) tienen un impacto adicional γ sobre la varianza condicional. Es el modelo estándar en riesgo de mercado profesional.
MLE — Maximum Likelihood Estimation
ℓ(θ) = Σ log f(εₜ | σₜ, ν)
θ̂ = argmax ℓ(θ) s.t. α+γ/2+β < 1
Estimamos ω, α, γ, β, ν simultáneamente maximizando la log-verosimilitud bajo Student-t. El parámetro ν se estima automáticamente — no se fija a priori.
Kupiec (1995) — POF Test
LR_uc = -2 log[(1-p)^(T-x) · p^x / ((1-x/T)^(T-x) · (x/T)^x)]
LR_uc ~ χ²(1) bajo H₀
Testa si el número de violaciones x es consistente con el nivel IC%. H₀: p(violación) = 1 - IC%. Rechazar H₀ → modelo mal calibrado.
Christoffersen (1998) — Independence Test
LR_ind = -2 log[L(π) / L(π₀₀, π₁₁)]
LR_cc = LR_uc + LR_ind ~ χ²(2)
Testa si las violaciones son independientes entre sí. Violaciones agrupadas (clústeres) indican que el modelo no captura la dinámica de volatilidad.
